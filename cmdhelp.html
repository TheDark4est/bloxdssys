死神永生指令系统

本服基于Bloxd API自研的指令系统，能够通过在聊天区输入指令，并完成一些Bloxd本不具备的功能，当前版本: 2.0

指令列表

#ssyscmd help

查看所有可用指令以及使用方法

#ssyscmd intro

获得死神永生服务器的简介

#ssyscmd illegals

获得所有可用的违禁物品，如绿宝石、砂石和干草等，请在使用前把背包清理出15个空位

#ssyscmd randtp

随机传送到一个位置，目前有bug，可能传送到天上或者地里，我会很快更新

#ssyscmd save 0 -401 [id] [text]

存储一段信息到死神永生硬盘。如果没有特殊用处，不要改0和-401两个数字。

[id]是自定义的整数，也是一条信息的唯一编号。如果和其他信息的id重复了，系统会给予提示，并修改为一个未使用的id。如果有提示，请以系统提示为准

请记住id，读取的时候要用到。

[text]是一段字符串，可以包含数字、字母和半角(英文)符号，目前不能存储中文

#ssyscmd read 0 -401 [id]

读取一条死神永生硬盘中的信息。如果没有特殊用处，不要改0和-401两个数字。

[id]是目标信息的号码，是一条信息的唯一编号。如果存储时，系统给了修改id的提示，请以系统提示为准

将在聊天区返回读取到的信息。
